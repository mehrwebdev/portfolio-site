# Portfolio Website

## Overview
This is my personal portfolio website to showcase my frontend projects, skills, and freelance work. It demonstrates responsive layouts and modern UI design.

## Features
- Fully responsive for desktop, tablet, and mobile
- Project showcase with images and descriptions
- About and Contact sections
- Interactive navigation

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
- Git & GitHub
- Figma (for design mockups, optional)

## How I Built It
- Planned the layout and structure (Home, About, Projects, Contact)
- Developed HTML structure with semantic elements
- Styled with CSS focusing on responsiveness and clean UI
- Added JavaScript for interactive elements
- Deployed on GitHub Pages

## What I Learned
- Creating responsive websites with Flexbox and Grid
- Structuring a professional portfolio
- Showcasing projects with good UX/UI
- Using Git and GitHub for version control
